## Configuration

In order to run the program, you may need to install additional software. Note that the below may or may not work for you.

```
sudo apt update
sudo apt install -y software-properties-common
sudo add-apt-repository -y universe
sudo apt update

sudo apt install -y dpdk libdpdk-dev dpdk-dev pkg-config
```
